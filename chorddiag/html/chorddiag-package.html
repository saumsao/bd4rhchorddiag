chorddiag: A package for creating interactive chord diagrams

Description

Create interactive chord diagrams using the JavaScript visualization library D3 (http://d3js.org). The chord diagram layout is based on http://bl.ocks.org/mbostock/4062006. Chord diagrams show directed relationships among a group of entities.
